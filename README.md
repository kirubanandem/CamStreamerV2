# CamStreamerV2
A pro level IP camera streamer and viewer.
Now convert your old or any android mobile to CCTV camera.
install the application.
connect to the internet.
start the dashboard.
start the stream and note the url (before staring setting up the password is recommended for web login)
note the streaming url.
connect using the app (Create profile, create cameras and enter the streaming url and password if set)
now enjoy your viewing...
battery percentage and temperature is monitored inside the app and web dashboard.
for static ip address to access over internet i'm recommending to use tailscale vpn network "https://tailscale.com/"
